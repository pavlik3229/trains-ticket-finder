# Beljd

Telegram-бот для отслеживания информации о поездах.

## Требования

- Python 3.12+
- Docker и Docker Compose — если хотите запускать в контейнере
- Файл `.env` в корне проекта с переменной:

```env
TG_TOKEN=ваш_токен_бота
```

## Установка локально

Создайте и активируйте виртуальное окружение:

```zsh
python3 -m venv .venv
source .venv/bin/activate
```

Установите зависимости:

```zsh
pip install -r requirements.txt
```

## Запуск локально

Запускать нужно как модуль из корня проекта:

```zsh
python -m src.main
```

## Запуск в Docker

Соберите и запустите контейнер через Compose:

```zsh
docker compose up --build
```

Если хотите запускать напрямую `docker run`, используйте образ, который собирается из `Dockerfile`, и передайте `.env` через `env_file` или `--env-file`.

## Что делает проект

- читает `TG_TOKEN` из `.env`
- запускает Telegram-бота
- периодически проверяет данные о поездах
- отправляет отчёты в чат пользователя

## Примечание

Проект использует `BeautifulSoup` с парсером `lxml`. Если при запуске будет ошибка, связанная с `lxml`, установите его отдельно или добавьте в зависимости проекта.
